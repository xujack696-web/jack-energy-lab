作品名称：Jack+ 的太空遨游实验舱

线上访问链接：
https://xujack696-web.github.io/jack-energy-lab/

作品说明：
本作品是人工智能理论与应用课程作业“我的数据小馆”个人主页。网页以“太空遨游实验舱”为主题，展示我的身份信息、音乐偏好、中国旅行足迹、志愿服务经历、互动实验室和航行能量系统。

主要功能：
1. 舰长身份识别卡：展示个人身份信息、个人头像、个人特质与头像吐槽交互。
2. 音乐补能舱：展示我喜欢的歌手与歌曲，并通过歌曲按钮、场景筛选和声波动画呈现音乐补能效果。
3. 中国坐标雷达：通过中国地图展示我的旅行足迹，点击城市点位可以查看对应记忆碎片。
4. 志愿星轨日志：以时间轴形式展示我的志愿服务经历。
5. 互动实验室：包含日夜航行模式切换、今日运动抽签和能量预测模拟器。
6. 航行能量舱：通过全局悬浮能量面板记录用户交互带来的航行能量变化。

文件说明：
index.html：网页主文件
assets/：网页使用的头像、歌手图片、中国地图等素材文件
README.txt：作品说明与线上部署链接
作业报告.pdf：作品设计说明、关键 Prompt 记录与踩坑复盘
